# 🛍️ Online Retail Sales Analysis

## 📌 Project Overview

This project presents an end-to-end retail sales analysis using Python and Power BI. The objective is to transform raw transactional data into meaningful business insights that support inventory management, sales optimization, customer analysis, and executive decision-making.

The project covers the complete analytics workflow, including data cleaning, feature engineering, exploratory data analysis, business insight generation, and interactive dashboard development.

---

# 📂 Dataset

**Dataset:** Online Retail Dataset

The dataset contains approximately one year of online retail transactions.

Each transaction includes:

- Invoice Number
- Product Code
- Product Description
- Quantity Purchased
- Invoice Date
- Unit Price
- Customer ID
- Country

---

# 🎯 Project Objectives

- Clean and preprocess retail transaction data.
- Handle missing values and inconsistent product descriptions.
- Perform exploratory data analysis.
- Analyze customer purchasing behaviour.
- Identify high-performing products.
- Study geographical sales performance.
- Perform Pareto Analysis (80/20 Rule).
- Perform ABC Inventory Classification.
- Build an interactive Power BI dashboard.

# 📈 Business Insights

The following business analyses were performed:

## ✅ Cancellation Analysis

- Measured cancellation rate.
- Identified cancellation patterns.

---

## ✅ Pareto Analysis

Applied the 80/20 principle to identify products responsible for the majority of company revenue.

---

## ✅ ABC Inventory Analysis

Products were classified into:

- **Category A** – Highest priority products
- **Category B** – Medium priority products
- **Category C** – Low priority products

This classification helps optimize inventory management.

# 💡 Key Business Recommendations

- Prioritize Category A products to maximize revenue.
- Maintain sufficient inventory for high-performing products.
- Investigate products with high cancellation rates.
- Increase marketing efforts for medium-performing products.
- Expand focus on top-performing countries.
- Monitor customer behaviour to improve retention.
